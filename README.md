# Projets Réseau Cisco Packet Tracer

Bienvenue sur ce repository ! Ici, vous trouverez des projets réseau créés avec **Cisco Packet Tracer** pour pratiquer et comprendre différents concepts de réseautique.

---

## Contenu du repository

| Fichier `.pkt` | Description |
|----------------|------------|
| `OSI.pkt` | Ce fichier Packet Tracer permet de visualiser le passage des données à travers les différentes couches du modèle OSI. |


---

## Instructions

1. Téléchargez le fichier `.pkt` de votre choix.
2. Ouvrez-le avec **Cisco Packet Tracer**.
3. Explorez le réseau, testez les configurations et modifiez-les pour apprendre.
4. Consultez les commentaires dans le projet pour comprendre chaque étape.

---

## Objectifs pédagogiques

- Comprendre le fonctionnement des routeurs et switches.
- Pratiquer le routage statique et dynamique (OSPF).
- Configurer des VLANs et tester la communication entre eux.
- Simuler un réseau complet et dépanner les problèmes de connectivité.

---

## Technologies utilisées

- Cisco Packet Tracer
- Protocoles : TCP/IP, OSPF, VLANs, DHCP, NAT
